# sberbank-garch-volatility-forecast
Построена GARCH(1,1) модель для прогнозирования волатильности акций Сбербанка (1999–2024). Выявлены ARCH-эффекты, сравнены модели с нормальным и t-распределением. Рассчитаны показатели VaR (95%: -3.69%, 99%: -6.46%). Модель с t-распределением показала лучшее качество (AIC ниже на 836 пунктов).
